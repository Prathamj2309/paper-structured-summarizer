\# Structured Academic Paper Summarizer



This project builds a transformer-based NLP system to generate structured summaries

(Contribution, Method, Limitations) from academic papers using retrieval-augmented generation (RAG).



\## Project Goals

\- Study transformer behavior on long-form scientific text

\- Generate structured summaries instead of free-form text

\- Analyze failure modes such as hallucination and repetition
